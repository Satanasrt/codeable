HTML

HTML es un lenguaje de marcado que se utiliza para crear páginas web. Es un estándar que se ha establecido en 1991 y es el lenguaje básico para la elaboración de páginas web.

Elementos de HTML

Los elementos de HTML son las partes del código HTML que definen el contenido de la página web. Existen diferentes tipos de elementos, algunos de ellos son:

  • Etiquetas de apertura: son las etiquetas que marcan el comienzo de un elemento, como <h1> para crear un título de primer nivel.
  • Etiquetas de cierre: son las etiquetas que marcan el final de un elemento, como </h
